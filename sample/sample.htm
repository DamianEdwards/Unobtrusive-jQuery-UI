Unobtrusive jQuery UI Example

This page shows an example of unobtrusively wiring-up jQuery UI to HTML elements.

To use the script on a page simply:

  1. Add a reference to jQuery

    <script src="script/jquery-1.7.1.js"></script>

  2. Add a reference to jQuery UI

    <script src="script/jquery-ui.1.8.17.js"></script>

  3. Add a reference to Unobtrusive jQuery UI

    <script src="script/jquery-ui.unobtrusive.js"></script>

  4. Add a link to a jQuery UI theme CSS file in the head

    <link rel="stylesheet" href="style/base/jquery.ui.all.css" />

  5. Add data-ui-fn attributes to the HTML elements you wish to add jQuery UI features to

    <input name="dateOfBirth" data-ui-fn="datepicker" /> <div data-ui-fn="draggable resizable"></div>

  6. Add further data-ui-* attributes to set options

    <div data-ui-fn="draggable" data-ui-draggable-containment="parent"></div>
